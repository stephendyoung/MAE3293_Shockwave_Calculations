To use these scripts:

1. Download or clone the GitHub repository to the same directory. 
2. Run the User_Input.py script.
3. Enter what appendix you are looking to pull values from. 
4. Enter mach number for the appendix. 
5. Read values at output.

The upstream pressure is only for the normal shockwave calculator and I am currently working to create a new method to only have it passed to certain methods in my class. Please ignore it for now or use it to your advantage.
